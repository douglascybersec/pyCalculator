## Installation & Usage:
1. Clone the repository:
```
git clone https://github.com/douglascybersec/pyCalculator.git

```

2. Navigate to project directory:
```
cd pyCalculator

```

3. Install dependencies:
```
pip install -r requirements.txt

```

4. Run the Application:
```
python pyCalc.py

```
